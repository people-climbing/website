# Product Overview

peopleclimbing.com is a PS2 menu-inspired interactive 3D website. It features a grid of 3D models that users can interact with, each linking to YouTube videos. The site uses a nostalgic aesthetic with starfield backgrounds, bloom effects, and rotating 3D objects positioned in an orthographic camera view reminiscent of PlayStation 2 system menus.

Key features:
- Interactive 3D model grid with hover and click interactions
- YouTube video overlay on model selection
- Real-time clock display
- Static site generation for GitHub Pages deployment
